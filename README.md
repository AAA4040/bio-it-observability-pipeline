# Bio-IT Observability Pipeline

This is a solution for a group of biology apps. It handles a **big amount of data per second**, then it filters the data, stores it in a database, and displays it on a dashboard.

Project Highlight: Real-time Medical Log Monitor

Built a customized Alpine-based Docker image for log observability.

Implemented a Bash script to monitor health metrics stream using tail and grep.

Configured Docker Bind Mounts for persistent data analysis between Host and Container.

Optimized streaming to handle only live events (zero-latency monitoring).
